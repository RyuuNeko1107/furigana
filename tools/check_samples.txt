# 検証ループ用「探索」例文集
#
# 使い方:
#   1. 新しい例文をここに足す (or 既存を書き換える)
#   2. `python3 tools/check_samples.py --binary <release-binary> --data-dir <staging>`
#      で出力を取得
#   3. 誤読を見つけたら lib (ja-furigana) または dict (ja-furigana-dict) を直す
#   4. 全部 OK になったら、その case を **ja-furigana-dict/tests/corpus/should_read.toml**
#      に `(input, mode, expected)` で **プロモート (アーカイブ)** する
#   5. ここのファイルは **次のラウンド用にリセット / 上書き** してよい
#      (回帰防止は CI の corpus job + should_read.toml が担う)
#
# つまり: ここは「探索の場」、保存場所は corpus 側。
# 過去ラウンドの 100 件は should_read.toml に追って promote 予定 (ROADMAP 参照)。

# M. 同形異音語 (heteronym) 特化 — 同じ漢字で文脈次第で読みが変わる
# 上手: ジョウズ (skill) / カミテ (舞台用語) / ウワテ
上手な料理人が上手から登場
彼の演技は上手で、観客を上手に魅了した

# 下手: ヘタ (poor) / シモテ (舞台用語) / シタテ (humble)

下手に出る彼の姿勢

# 一月 / 二月: 暦 (イチガツ / ニガツ) vs 期間 (ヒトツキ / フタツキ)

一月ほど待ってください、二月かかるでしょう

# 市場: シジョウ (金融・抽象) / イチバ (実物市場)

# 十分: ジュウブン (sufficient) / ジュップン (10 min)
